
### 表达式格式符号

[ ] —— 条件测试命令，只可以用-eq之类的，且变量前必须要$
(( )) —— 算术运算与算术条件，可以用>等数学运算表达式，变量不需要带$,且赋值(例如i++)的时候一定不能带$


$(( )) 用于算术展开，把计算结果替换到命令行中（例如 sum=$((a+b))）。

[[ ]]	增强条件测试（Bash）	==, =~, -gt	正则匹配（独有！！！）、逻辑组合、更安全

#### 例子
var="a b c"
[ $var = "a b c" ]   # 错误！被拆成 [ a b c = "a b c" ] → 太多参数
[ "$var" = "a b c" ] # 正确，必须加引号

[[ $var = "a b c" ]] # 正确，无需引号


file="*.txt"
[ $file = "*.txt" ]   # 如果当前目录有 a.txt b.txt，会扩展成多个参数，出错
[ "$file" = "*.txt" ] # 正确

[[ $file = "*.txt" ]] # 正确


#### 用 [ ] 时，推荐外部组合：
[ "$a" -gt 0 ] && [ "$b" -lt 10 ]

#### 或者用 -a（不推荐，易被误解）
[ "$a" -gt 0 -a "$b" -lt 10 ]

#### 用 [[ ]] 可以直接写：
[[ a -gt 0 && b -lt 10 ]]
#### 甚至数值比较可以用 (( ))：
(( a > 0 && b < 10 ))


### sed

一次处理文件的一行，处理完echo在屏幕


-n 屏蔽默认输出一行，默认sed会输出文档的全部内容
-r 支持正则表达式//，默认支持
-i 直接修改源文件
-e 执行多行命令，类似于管道符，可以继续接子操作，不过第一个子操作就要加-e了；注意-e与其他选项是平级的，只要有-i，那就一定会直接修改源文件，不针对-i是具体加在那个-e后面


子操作命令
p 查看 -n
d 删除 不能-n,默认是把
s 查找替换 s/old/new/选项
[n]a第n行后面插入
[n]i第n行前面插入
[n]c替换


#### p
sed -n 'p' file.name   等于cat
sed -n '4p' file.name   查看第四行
sed -n '4,7p' file.name   查看第四-七行
sed -n '/^do/p' file.name   查看以do开头的所有行
sed -n '$=' file.name   输出文件行数

#### d
sed '3,5d' file.name 删除3-5行并查看
sed '/xml/d' file.name 删除包含xml的行
sed '/xml/!d' file.name 删除不包含xml的行
sed '/^$/d' file.name 删除空行 ，^代表开头,$代表结尾

#### s


sed 's/xml/XML/'  a.txt      //将每行中第一个xml替换为XML
sed 's/xml/XML/3' a.txt     //将每行中的第3个xml替换为XML
sed 's/xml/XML/g' a.txt     //将所有的xml都替换为XML
sed 's/xml//g'     a.txt     //将所有的xml都删除（替换为空串）
sed 's#/bin/bash#/sbin/sh#' a.txt  //将/bin/bash替换为/sbin/sh，#用来转义/符号
sed '4,7s/^/#/'   a.txt      //将第4~7行注释掉（行首加#号）
sed 's/^#an/an/'  a.txt      //解除以#an开头的行的注释（去除行首的#号）

注意：‘g’表示global，全部替换；数字在前表示行，数字在后表示第几个




### awk

awk [] 'BEGIN {commands} pattern {commands}  END{commands}'

commands没有echo，只有print?

-F 分隔符
-v 变量
BEGIN 加表头
END 加表尾

awk -F : 'BEGIN{print "A,B,C"} {print $1,$7} END{print "dahaige /bin/zuishuai"}' /etc/passwd

awk -F : 'BEGIN{print "A,B,C"} {print $1","$7} END{print "dahaige,/bin/zuishuai"}' /etc/passwd

//输出用户ID+1
awk -v i=1 -F: '{print $3+i}' /etc/passwd

#### 内置的变量
FILENAME	文件名
NR	已读的记录数
NF	浏览记录的域的个数（切割后，列的个数）

查找空行
awk '/^$/{print NR}' cut.txt

查找以空格开头空格结尾的行
awk '/^ .* $/{print NR}' text_sample.txt
