### 权限

执行权限是文件的最高权限
写权限是目录的最高权限（删除目录下的所有对象，无论是否有对子目录或者文件的权限）

### 用户组
groupadd
```
不-g指定，默认会新建一个组ID（当前最大组ID+1）
groupadd -g <groupname>
```

/etc/group
```
x是密码口号，一般都没有密码
第四列是隶属于该组的用户
```

groupdel
```
groupdel <groupname>
```

修改文件归属的组
chgrp
```
chgrp <groupname> <filename>
chgrp -R <groupname> <dirname> #递归修改
```

### chmod
chmod a/u/g/o   +/-  r/w/x
chmod a/u/g/o   =  r/w/x/

错误写法：chmod u=r--,g=-w-,o=--x 文件名
正确写法：chmod u=r,g=w,o=x 文件名 （或者 chmod u=r,g=w,o=x 文件名）
